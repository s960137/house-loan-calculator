# 🏠 房貸試算互動工具 (Mortgage Calculator)

**房貸試算小幫手**！這是為「想買房但怕算數」設計的工具。
**手機也能支援**！

## 🌟 專案特色 (Features)

不只能算數學，還幫數字變成圖表

- **🛠 自定義參數**：自由輸入房屋總價、貸款成數、年利率與寬限期。
- **📊 雙模式比較**：一鍵切換「本息平均攤還」與「本金平均攤還」，看哪種省錢。
- **📈 視覺化圖表**：使用 Chart.js 繪製每月繳款變化圖。
- **📱 手機優化**：針對行動裝置設計，按鈕好按、字體清晰。


##  使用技術 (Tech Stack)

- **HTML5**：建構網頁骨架。
- **Tailwind CSS (CDN)**：不需 CSS。
- **Chart.js**：圖表庫，繪製折線圖。
- **Vanilla JavaScript**： JS 撰寫邏輯無多餘依賴。
